# How to run

e.g. `MyReactNative@0.69.4`

```sh
$ cd MyReactNative@0.69.4
```

e.g. `MyReactNative@0.69.4`

```sh
$ npx react-native@0.69.4 init MyReactNative --version 0.69.4
$ mv MyReactNative MyReactNative@0.69.4
```
